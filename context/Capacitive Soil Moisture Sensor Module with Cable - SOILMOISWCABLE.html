With this sensor you measure the humidity of the soil. The sensor distinguishes itself from other soil moisture sensors by the capacitive measurement instead of a resistance measurement. This makes the sensor more resistant to corrosion which is why it will last longer.



Specifications:



    
	
  • Input voltage: DC 3.3 - 5V
    • 
	
  • Output voltage: DC 0 - 3.0V
    • 
	
  • Size: 99x16mm
    • 




Comes with a 3-pin cable.
